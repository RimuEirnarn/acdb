# ACDB - Anime Character Database Web Scraping

This is one of unofficial ACDB scraping scripts.

## Install

```sh
git clone https://github.com/RimuEirnarn/acdb
cd acdb
pip install -r ./requirements.txt
python3 acdb.py
```

## Usage

```sh
python3 acdb.py -m <month> -d <day>
```
